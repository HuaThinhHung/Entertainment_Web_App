## 🚀 Tính Năng Nổi Bật

- 🎬 Hiển thị dữ liệu phim theo thời gian thực từ **TMDB API**
- 🔍 Tìm kiếm thông minh, hiển thị gợi ý ngay khi gõ
- 📑 Lưu phim yêu thích (Bookmark) với LocalStorage
- 🌙 Chuyển đổi chế độ Sáng/Tối mượt mà
- 📱 Giao diện tương thích mọi thiết bị (responsive)
- ⚡ Hiệu suất cao, hoạt ảnh mượt
- 💾 Hỗ trợ ngoại tuyến (offline) với LocalStorage
